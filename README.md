# 凯撒密码加密/解密工具

一个基于 Python tkinter 的图形化凯撒密码工具，支持加密和解密功能。

## 功能特点

- 🎨 简洁直观的图形界面
- 🔐 支持凯撒密码加密和解密
- ⌨️ 处理大小写字母和特殊字符
- ⚡ 实时状态提示
- 🧹 一键清空功能



## 使用方法

1. 在明文/密文区域输入文本
2. 设置密钥（1-25之间的整数）
3. 点击"加密"或"解密"按钮
4. 查看结果区域获取处理后的文本

## 运行要求

- Python 3.x
- tkinter（通常随 Python 一起安装）
